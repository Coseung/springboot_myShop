# Spring Boot 개인 공부 프로젝트 🚀

 **Spring Boot**를 사용한 개발 공부. 다양한 기능을 구현하며 웹 애플리케이션의 핵심 요소들을 다뤄보는 시간을 가졌다. 프로젝트는 AWS Elastic Beanstalk를 통해 배포되었으며, 여러 기술 스택을 활용.

## 주요 기능 🔧

- **세션 기반 로그인 기능** 🔐
- **DTO를 활용한 데이터 전송** 📦
- **페이지네이션 구현** 📄
- **AWS S3를 이용한 이미지 업로드** 🌄
- **AWS Elastic Beanstalk를 통한 서버 배포** ☁️
- **댓글 기능** 💬
- **검색 기능** 🔍
- **주문 기능** 🛒
